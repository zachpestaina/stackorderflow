# stackorderflow

A web scraping tool designed to enable users to save and associate Stack Overflow articles and top answers with their current projects.

<h3> Technologies used in the project </h3>

<ul>
  <li> Cheerio </li>
  <li> React Hooks </li>
  <li> Node.js </li>
  <li> OAuth 2.0 </li>
</ul>
